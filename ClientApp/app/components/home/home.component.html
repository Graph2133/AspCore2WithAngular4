HI!

Welcome to single-page application, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • angular2-jwt for user authorization
  • Bootstrap for layout and styling
  • ng2-toasty for server message diplaying
  • Webpack for building and bundling client-side resources
  • Entity Framework Core to work with a database using .NET objects
  • FluentValidation for dto's validation
  • ASP.NET Core Identity membership system that adds login functionality to ASP.NET Core apps (Role-based JWT authentication scheme was used)
